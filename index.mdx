## Five modules, one import

```python
from axion import Axion        # REST API client
from axion import ta           # Technical analysis indicators
from axion import visualize    # Interactive Plotly charts
from axion import utils        # DataFrame & date utilities
from axion import models       # ML / forecasting models
```

## Quick example

```python quickstart.py
from axion import Axion, ta, visualize, utils as axion_utils

# Initialize the client
client = Axion(api_key="your_api_key_here")

# Fetch daily prices for Apple
prices = client.stocks.prices("AAPL", from_date="2024-01-01")
df = axion_utils.df(prices)

# Compute a 14-period RSI
df["rsi"] = ta.rsi(df, column="close", period=14)

# Render a candlestick chart
visualize.candles(df)
```

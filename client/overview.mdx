The `Axion` client is the single entry point to every API namespace in the SDK. Instantiate it once with your API key and access any data domain through its attributes.

```python
from axion import Axion

client = Axion(api_key="your_api_key_here")
```

<Note>
  Your API key is sent as a `Bearer` token in the `Authorization` header on every authenticated request. Keep it secure and never commit it to source control.
</Note>

## Error handling

All methods raise a plain `Exception` with a descriptive message on failure. The message includes the HTTP status code and the API error message where available.

```python
try:
    data = client.stocks.ticker("INVALID")
except Exception as e:
    print(e)  # HTTP Error 404: Ticker not found
```

<Tip>
  Wrap API calls in `try/except` blocks in production code. Network errors, timeouts, and HTTP errors are all surfaced as `Exception` instances.
</Tip>

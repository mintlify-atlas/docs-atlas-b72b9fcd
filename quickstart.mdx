## Prerequisites

- Python 3.8 or higher
- An Axion API key — [get one free](https://axionquant.com/dashboard/api-keys)

## Install the SDK

```bash
pip install axionquant-sdk
```

## Initialize the client

```python
from axion import Axion

client = Axion(api_key="your_api_key_here")
```

<Tip>
  Store your API key in an environment variable rather than hardcoding it. See [Authentication](/guides/authentication) for the recommended setup.
</Tip>

## Fetch your first data

<Steps>
  <Step title="Get stock prices">
    Fetch daily OHLCV data for Apple (AAPL) from the start of 2024:

    ```python
    prices = client.stocks.prices("AAPL", from_date="2024-01-01")
    ```

    This returns a list of dicts, one per trading day.
  </Step>

  <Step title="Convert to a DataFrame">
    Use the `utils.df()` helper to turn the response into a pandas DataFrame:

    ```python
    from axion import utils as axion_utils

    df = axion_utils.df(prices)
    print(df.head())
    ```

    ```
           time     open     high      low    close      volume
    0  2024-01-02  185.07  186.00  183.43  184.25  82702900
    1  2024-01-03  183.81  184.81  182.00  183.52  55796000
    ...
    ```
  </Step>

  <Step title="Run a technical indicator">
    Compute a 14-period RSI on the closing prices:

    ```python
    from axion import ta

    df["rsi"] = ta.rsi(df, column="close", period=14)
    print(df[["time", "close", "rsi"]].tail())
    ```
  </Step>

  <Step title="Visualize the data">
    Render an interactive candlestick chart:

    ```python
    from axion import visualize

    visualize.candles(df)
    ```

    The chart opens inline in Jupyter or in your default browser.
  </Step>
</Steps>

## Full working example

```python full_example.py
from axion import Axion, ta, visualize, utils as axion_utils

# Initialize
client = Axion(api_key="your_api_key_here")

# Fetch prices
prices = client.stocks.prices("AAPL", from_date="2024-01-01", to_date="2024-12-31")
df = axion_utils.df(prices)

# Add RSI and MACD
df["rsi"] = ta.rsi(df, column="close", period=14)
macd_line, signal, histogram = ta.macd(df)
df["macd"] = macd_line
df["signal"] = signal

# Visualize
visualize.candles(df)
visualize.graph(df, x="time", bars=["volume"], lines=["close", "macd", "signal"])
```
